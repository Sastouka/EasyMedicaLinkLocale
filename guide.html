Guide d'Utilisation

Bienvenue dans la documentation officielle. Maîtrisez EasyMedicaLink en quelques minutes.

Installation & Démarrage

EasyMedicaLink utilise une architecture hybride (Serveur/Client). Voici comment configurer votre cabinet.

Étape 1 : Le PC "Serveur" Choisissez l'ordinateur le plus puissant du cabinet (celui de l'accueil ou du médecin). Téléchargez et installez la version x64 (recommandée).
Étape 2 : L'Assistant d'Installation Lancez l'installateur. Cliquez sur "Suivant" jusqu'à la fin. Une icône "EasyMedicaLink Server" apparaîtra sur le bureau.
Astuce : Désactivez la mise en veille sur le PC Serveur pour garantir un accès continu aux autres appareils (tablettes, secrétariat).

Connexion

Au premier lancement, le logiciel vous demandera de créer le compte Administrateur.

  • Utilisateur par défaut : admin
  • Mot de passe : (Celui que vous définirez)

Une fois connecté, allez dans Paramètres > Clinique pour entrer le nom de votre cabinet et votre logo pour les ordonnances.

Agenda & RDV

Le module RDV est le cœur du secrétariat. Il permet de gérer le flux de patients.

Ajouter un RDV Cliquez sur une case horaire libre. Recherchez un patient existant ou créez-en un "Rapide" (Nom + Tél uniquement).
Salle d'attente Dès que le patient arrive, faites un clic droit sur le RDV > "Marquer comme Arrivé". Il apparaîtra instantanément sur l'écran du médecin.

La Consultation

Interface optimisée pour la saisie rapide durant l'examen.

  • Constantes : Saisie rapide (Tension, Poids, Température).
  • Ordonnance : Tapez les 3 premières lettres d'un médicament. La posologie par défaut s'affiche automatiquement.
  • Documents : Cliquez sur "Certificat" ou "Lettre" pour générer un PDF prêt à imprimer.

Assistant IA (Synapse)

Utilisez l'intelligence artificielle pour analyser des cas complexes.

Analyse de documents Vous pouvez scanner un rapport de biologie (PDF ou Image) et demander à l'IA : "Résume-moi les anomalies de ce bilan sanguin".
Important : L'IA est une aide à la décision. Vérifiez toujours ses suggestions.

Connexion des autres appareils

Pour connecter une tablette ou un Mac :

  1. Assurez-vous d'être sur le même réseau Wi-Fi que le PC Serveur.
  2. Sur le PC Serveur, regardez en bas de l'écran d'accueil, l'adresse IP est affichée (ex: http://192.168.1.15:3000).
  3. Ouvrez Chrome ou Safari sur votre tablette et tapez cette adresse.
